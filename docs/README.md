## 📖 文档命名说明

为了兼容性，文档文件名使用英文命名。以下是文件名与中文含义的对应关系：

### 架构设计文档
- `Architecture_Overview.md` → 架构总览
- `Architecture_RHI.md` → RHI架构设计
- `Architecture_Refactoring.md` → 架构重构记录
- `Code_Analysis.md` → 代码分析
- `Modules.md` → 模块说明
- `VS_Solution_Structure.md` → VisualStudio解决方案结构

### Bug修复文档
所有Bug修复文档都在 `BugFixes/` 目录下，文件名采用 `问题类型_修复方式.md` 的格式。

### 模块文档
模块文档按模块分类，文件名格式为 `模块名_文档类型.md`。

## 🔍 快速查找

### 按问题类型查找
- **DLL问题** → `BugFixes/DLL_*`
- **调试问题** → `BugFixes/*Debug*` 或 `BugFixes/*Breakpoint*`
- **编辑器问题** → `BugFixes/Editor_*`

### 按模块查找
- **核心模块** → `Modules/Core/`
- **渲染模块** → `Modules/Renderer/`
- **资源模块** → `Modules/Resources/`
- **着色器模块** → `Modules/Shader/`
- **编辑器模块** → `Modules/Editor/`
