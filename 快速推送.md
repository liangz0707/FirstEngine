# 快速推送到 GitHub

## 如果你已经有 GitHub 仓库

### 使用脚本（推荐）
```powershell
.\push_to_github.ps1 -GitHubUsername YOUR_USERNAME -RepoName FirstEngine
```

### 手动命令
```powershell
# 添加远程仓库（替换 YOUR_USERNAME）
git remote add origin https://github.com/YOUR_USERNAME/FirstEngine.git

# 推送到GitHub
git push -u origin main
```

## 如果还没有创建 GitHub 仓库

1. **访问 https://github.com** 并登录
2. **点击右上角 "+" → "New repository"**
3. **填写信息**：
   - Repository name: `FirstEngine`
   - Description: `Modern 3D Graphics Engine built with C++, Vulkan, and GLFW`
   - 选择 Public 或 Private
   - **不要**勾选任何初始化选项
4. **点击 "Create repository"**
5. **然后运行上面的命令**

## 认证说明

如果推送时要求输入密码：
- **用户名**：你的GitHub用户名
- **密码**：需要使用 **Personal Access Token**（不是GitHub密码）

### 创建 Personal Access Token
1. 访问：https://github.com/settings/tokens
2. 点击 "Generate new token" → "Generate new token (classic)"
3. 勾选 `repo` 权限
4. 点击 "Generate token"
5. **复制生成的token**（只显示一次！）
6. 推送时在密码输入框粘贴这个token
